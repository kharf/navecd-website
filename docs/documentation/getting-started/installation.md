!!! note

    Currently we don't maintain our binaries in any package manager.

You can install Navecd by downloading the binaries directly:

=== "Linux (x86_64)"

    ``` bash title="Install script"
    curl -L -o navecd.tar.gz https://github.com/kharf/navecd/releases/download/v0.24.10/navecd_linux_x86_64.tar.gz
    tar -xf navecd.tar.gz
    chmod +x navecd
    ./navecd -h
    ```

=== "MacOS (x86_64)"

    ``` bash title="Install script"
    curl -L -o navecd.tar.gz https://github.com/kharf/navecd/releases/download/v0.24.10/navecd_darwin_x86_64.tar.gz
    ```
